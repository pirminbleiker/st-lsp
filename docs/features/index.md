ST LSP provides a full set of language intelligence features for IEC 61131-3 Structured Text.

|---------|-------------|
| [Code Completion](completion.md) | Keyword, type, FB, and variable suggestions |
| [Diagnostics](diagnostics.md) | Real-time parse error reporting |
| [Navigation](navigation.md) | Hover documentation and Go-to-Definition |
